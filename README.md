# 🎭 Impostor Game

A modern single-phone pass-device social deduction game built with HTML, CSS, and JavaScript.

Players take turns revealing their secret role on the same device:
- Most players receive the same word
- One player becomes the IMPOSTOR 👀

The impostor must blend in during discussion without knowing the secret word.

---

# ✨ Features

- 📱 Single-phone pass-device gameplay
- 🎭 Random impostor system
- 🔥 Random word generator
- 🕵️ Secret role reveal flow
- 💎 Modern glassmorphism UI
- 🌙 Mobile-friendly design
- ⚡ Fast lightweight gameplay
- 🔁 Replay support

---

# 🎮 How To Play

1. Select number of players
2. Pass the device to each player
3. Reveal your role secretly
4. Discuss with the group
5. Find the impostor before time runs out

---

# 🛠️ Built With

- HTML
- CSS
- JavaScript

No backend required.
